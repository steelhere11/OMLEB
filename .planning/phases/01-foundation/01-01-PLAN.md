       - id (uuid, PK, default gen_random_uuid())

       - created_at (timestamptz, default now())
       - updated_at (timestamptz, default now())

       **sucursales**
       - id (uuid, PK, default gen_random_uuid())
       - nombre (text, NOT NULL)
       - numero (text, NOT NULL)
       - direccion (text, NOT NULL)
       - created_at (timestamptz, default now())
       - updated_at (timestamptz, default now())

       **equipos**
       - id (uuid, PK, default gen_random_uuid())
       - sucursal_id (uuid, FK -> sucursales, NOT NULL)
       - numero_etiqueta (text, NOT NULL)
       - marca (text)
       - modelo (text)
       - numero_serie (text)
       - tipo_equipo (text)
       - agregado_por (uuid, FK -> users, nullable)
       - revisado (boolean, default false)
       - created_at (timestamptz, default now())
       - updated_at (timestamptz, default now())

       **folios**
       - id (uuid, PK, default gen_random_uuid())
       - sucursal_id (uuid, FK -> sucursales, NOT NULL)
       - cliente_id (uuid, FK -> clientes, NOT NULL)
       - numero_folio (text, NOT NULL, UNIQUE, auto-generated via trigger or sequence)
       - descripcion_problema (text, NOT NULL)
       - estatus (text, NOT NULL, CHECK: 'abierto', 'en_progreso', 'completado', 'en_espera', default 'abierto')
       - created_at (timestamptz, default now())
       - updated_at (timestamptz, default now())

       **folio_asignados**
       - id (uuid, PK, default gen_random_uuid())
       - folio_id (uuid, FK -> folios, NOT NULL)
       - usuario_id (uuid, FK -> users, NOT NULL)
       - created_at (timestamptz, default now())

       **reportes**
       - id (uuid, PK, default gen_random_uuid())

       - firma_encargado (text, nullable)
       - finalizado_por_admin (boolean, default false)
       - created_at (timestamptz, default now())
       - updated_at (timestamptz, default now())

       **reporte_equipos**
       - id (uuid, PK, default gen_random_uuid())
       - reporte_id (uuid, FK -> reportes, NOT NULL, ON DELETE CASCADE)
       - equipo_id (uuid, FK -> equipos, NOT NULL)
       - tipo_trabajo (text, NOT NULL, CHECK: 'preventivo', 'correctivo')
       - diagnostico (text)
       - trabajo_realizado (text)
       - observaciones (text)

       - cantidad (numeric, NOT NULL)

       - descripcion (text, NOT NULL)

       Add the `handle_new_user()` trigger from RESEARCH.md (Pattern 4) that syncs auth.users inserts to public.users. Include `ON CONFLICT (id) DO NOTHING`.

       Add a folio numbering trigger or sequence: auto-generate `numero_folio` as a padded sequential string (e.g., "F-0001", "F-0002"). Use a Postgres sequence.

    2. Create `supabase/rls.sql` with comprehensive RLS policies:

       First, create the `private` schema and SECURITY DEFINER helper functions (from RESEARCH.md):
       - `private.get_user_role()` -- returns current user's rol from JWT app_metadata
       - `private.is_admin()` -- returns boolean
       - `private.get_my_folio_ids()` -- returns folio IDs assigned to current user (avoids circular RLS)

       IMPORTANT: Wrap all `auth.uid()` and `auth.jwt()` calls in `(SELECT ...)` for performance (94%+ improvement per Supabase docs).

       Then create RLS policies for ALL tables:

       **users**: Admin can read all. Technicians can read all (needed to see team members). Only admin can insert (via service role for creation). Users can update their own row (nombre only).

       **clientes**: Admin full CRUD. Technicians read-only (needed to see client info on folios).

       **sucursales**: Admin full CRUD. Technicians read-only.

       **equipos**: Admin full CRUD. Technicians can read (branch equipment) and insert (field-added equipment, sets agregado_por to own id).

       **folios**: Admin full CRUD. Technicians can read assigned folios only (use `private.get_my_folio_ids()`).

       **folio_asignados**: Admin full CRUD. Technicians can read own assignments.

       **reportes**: Admin full CRUD. Technicians can CRUD on reports for their assigned folios (use `private.get_my_folio_ids()`).

       **reporte_equipos**: Same as reportes -- admin full, technicians via folio assignment.

       **reporte_fotos**: Same as reportes.

       **reporte_materiales**: Same as reportes.

       Enable RLS on ALL tables with `ALTER TABLE ... ENABLE ROW LEVEL SECURITY;`
