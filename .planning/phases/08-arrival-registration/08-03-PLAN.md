    Photo capture flow (follows existing pattern in workflow-step-card.tsx):
    1. User taps "Tomar foto de llegada"

    5. On success: call `completeArrival(reporteId)` server action (import from `@/app/actions/registration`)
    6. Update local state with photo URL
    7. Call `onComplete()` callback to notify parent

    If Gallery: file input, same upload flow but skip CameraCapture.

    GPS acquisition: Use `navigator.geolocation.getCurrentPosition` on mount to get position. Store as `lat,lng` string. Same pattern as existing CameraCapture.
