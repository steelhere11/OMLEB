# Plan 09-06: Technician-Side Feedback Visibility

<objective>
Make admin feedback visible to technicians — flagged photos with admin notes, admin comments, and a prominent banner showing items that need attention (retomar photos, rejected items).
</objective>

## Context

Plans 09-03 and 09-05 created photo flagging (estatus_revision, nota_admin) and admin comments (reporte_comentarios). Technicians have RLS read access to comments. This plan adds the technician-side UI to display this feedback.

### Current Technician Report View
The technician report page at `src/app/tecnico/reporte/[reporteId]/page.tsx` shows the report form with equipment sections, evidence stages, materials, etc. We need to enhance it with admin feedback visibility.

### Key Principle: Feedback Loop
Admin flags photo as "retomar" with note "Foto borrosa, retomar desde mas cerca" → Technician opens report → sees banner "1 foto necesita ser retomada" → taps to go to that step → sees the admin note → retakes the photo.

## Tasks

### Task 1: Create AdminFeedbackBanner component
**File:** `src/components/tecnico/admin-feedback-banner.tsx` (NEW, ~60 LOC)

A prominent banner shown at the top of the technician's report page when there are items needing attention.

Props:
```typescript
interface AdminFeedbackBannerProps {
  retakePhotos: Array<{
    fotoId: string;
    etiqueta: string;
    equipoLabel: string;
    stepName?: string;
    nota: string | null;
  }>;
  rejectedPhotos: Array<{
    fotoId: string;
    etiqueta: string;
    equipoLabel: string;
    nota: string | null;
  }>;
  commentCount: number;
}
```

UI:
1. Only renders if there are retake/rejected items or comments
2. Yellow/amber banner with warning icon
3. If retake photos: "X foto(s) necesitan ser retomadas" with list:
   - "{equipoLabel} - {stepName}: {nota}" for each
4. If rejected photos: "X foto(s) rechazadas" with list
5. If comments: "X comentario(s) del administrador"
6. Each item is tappable — scrolls to the relevant section (optional, or just informational)
7. Dismissable with X button (but reappears on next visit if unresolved)

All text in Spanish.
