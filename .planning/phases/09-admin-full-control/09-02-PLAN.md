# Plan 09-02: Delete UI — Folios, Reportes, Equipos, Sucursales

<objective>
Add cascade delete UI across all admin entity pages. Create a reusable CascadeDeleteButton component that shows impact summary and requires typed confirmation for high-impact deletes.
</objective>

## Context

Plan 09-01 created all cascade delete server actions (adminDeleteReport, adminDeleteFolio, adminDeleteEquipo, adminDeleteSucursal). This plan adds the UI to trigger them.

### Impact Summary Pattern
Before confirming a delete, show the user what will be deleted:
- "Este folio tiene 3 reportes con 47 fotos. Se eliminaran todos."
- For high-impact deletes (folios with reports, sucursales with data), require typing the entity name/number.

## Tasks

### Task 1: Create CascadeDeleteButton component
**File:** `src/components/admin/cascade-delete-button.tsx` (NEW, ~80 LOC)

Props:
```typescript
interface CascadeDeleteButtonProps {
  entityType: 'folio' | 'reporte' | 'equipo' | 'sucursal';
  entityId: string;
  entityLabel: string; // Display name for confirmation
  onDelete: (id: string) => Promise<{ success: boolean; error?: string }>;
  impactSummary?: string; // e.g., "3 reportes, 47 fotos"
  requireTypedConfirmation?: boolean; // true for high-impact
  redirectTo?: string; // URL to redirect after successful delete
}
```

Behavior:
1. Red "Eliminar" button (small, not prominent)
2. On click: opens confirmation dialog
3. Dialog shows: "Esta a punto de eliminar {entityType} '{entityLabel}'"
4. If impactSummary: shows "Se eliminaran: {impactSummary}"
5. If requireTypedConfirmation: shows text input "Escriba '{entityLabel}' para confirmar"
6. Confirm button disabled until text matches (if typed confirmation required)
7. On confirm: calls onDelete, shows loading spinner
8. On success: redirects if redirectTo provided, otherwise revalidates
9. On error: shows error message in dialog
10. All text in Spanish

Use existing UI patterns from the codebase — check `src/components/ui/` for modal/dialog patterns. If there's no dialog component, use a simple overlay div with backdrop.
