    **src/components/shared/evidence-stage-section.tsx (NEW):**

    Create a shared component with interface:
    ```typescript
    interface EvidenceStageSectionProps {
      evidencia: { etapa: string; descripcion: string }[];
      photos: ReporteFoto[];
      onTakePhoto: (stage: string) => void;
      onDeletePhoto?: (fotoId: string) => void;
      isUploading?: boolean;
      activeStage?: string | null;
      disabled?: boolean;
    }
    ```

    Implementation:
    - Extract unique stages from evidencia: `Array.from(new Set(evidencia.map(ev => ev.etapa)))`

    - For each stage, collect descriptions: `evidencia.filter(ev => ev.etapa === stage).map(ev => ev.descripcion)`
    - Render one section per stage with:

      - Camera icon + stage label ("ANTES" / "DURANTE" / "DESPUES")
      - Description bullets as `<p>` text elements (NOT buttons), styled as guidance text (text-xs text-gray-500)
      - Single "Tomar foto" button that triggers `onTakePhoto(stage)`
      - Photo count badge next to button (e.g., "3 fotos")
      - PhotoGalleryRow filtered to `photos.filter(p => p.etiqueta === stage)`, with delete handler if not disabled

    Layout structure per stage:
    ```
    ┌─ Stage Section ─────────────────────────────────┐
    │  Color header: "📷 ANTES" (blue)                │
    │                                                  │
    │  Guidance bullets:                               │

    │                                                  │
    │  [thumb] [thumb] [thumb]  (PhotoGalleryRow)      │
    └──────────────────────────────────────────────────┘
    ```
